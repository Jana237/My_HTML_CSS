в предыдущих версиях CSS разработчики были вынуждены использовать только те шрифты, которые гарантированно установлены на компьторе пользователя, в CSS разработчики могут использовать любые шрифты, которые они захотят. Когда вы найдете необходимый шрифт просто разместите его на веб-сервере и подключите его с помощью нового CSS3 свойства @font-face. Подключенный шрифт будет скачан и отображен автоматически при загрузки страницы.

обратите внимание: Internet Explorer поддерживает шрифты в формате .eot, остальные браузеры в формате .ttf т.е. для того , чтобы шрифт отображался корректно во всех браузерах Вы должны подключить его к странице в двух форматах.
